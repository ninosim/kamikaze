# Instalação

Passos para instalar e configurar o projeto:

### Clone o repositório
> git clone https://github.com/natanfiuza/help_rs_kamikaze.git

### Navegue até o diretório do projeto
> cd seu-projeto

### Instale as dependências
> pip install -r requirements.txt
